# MapaMania
Um jogo feito no penúltimo semestre da faculdade com o intuito de ajudar crianças com a matéria de geografia em uma escola que frequentamos com o problema na matéria.
O jogo foi feito no site editor.contruct, onde se usa apenas lógica para programar, apesar de ser muito mais fácil de se aprender e colocar em prática se souber linguagens como python.
Foi feito um Diagrama com relação de entidades e muito mais no Miro para termos como base todo o projeto, ele se encontra no seguinte link: https://miro.com/app/board/uXjVJL94Q2c=/
